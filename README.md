# Trabajo Práctico - Gestión Colaborativa, GitHub y Jira

## Descripción del Proyecto

Este proyecto fue desarrollado utilizando Google Colab, Python, GitHub y Jira con el objetivo de realizar un análisis de datos sobre vehículos usados.

Se trabajocon un dataset en formato Excel que fue procesado, limpiado y analizado mediante Python y la librería Pandas

---

## Tecnologías Utilizadas

* Python
* Google Colab
* Pandas
* Matplotlib
* GitHub
* Jira

---

# Objetivos del Trabajo

* Procesar datos desde archivos Excel.
* Limpiar y normalizar información.
* Generar grficos estadísticos
* Obtener conclusiones automáticas del dataset.
* Utilizar GitHub como sistema de control de versiones.
* Organizar tareas mediante Jira.

---

## Archivos del Proyecto

* Trabajo_Practico_Colab.ipynb

* Notebook principal desarrollado en Google Colab.

* dataset_limpio.xlsx

* Dataset procesado y normalizado.

* conclusiones.txt

* Resultados y conclusiones automáticas obtenidas del análisis.

---

## Análisis Realizado

Durante el proyecto se realizron las siguientes tareas:

1. Lectura del archivo Excel.
2. Exploración de hojas del dataset.
3. Limpieza de columnas innecesarias.
4. Normalización de datos.
5. Generación de gráficos estadísticos.
6. Obtención de métricas automáticas:

   * Cantidad total de registros.
   * Precio promedio.
   * Precio máximo.
   * Tipo de vehículo más frecuente

---

## Gestión del Proyecto

La organización de tareas se realizó mediante Jira utilizando un tablero Scrum.

Las tareas principales fueron:

* Configuración inicial del repositorio.
* Desarrollo de scripts y carga de datos.
* Integración GitHub y control de versiones

---

# Conclusión 

El trabajo permitió aplicar conceptos de análisis de datos, programación en Python, trabajo colaborativo y control de versiones utilizando herramientas profesionales utilizadas en entornos reales de desarrollo.
